UA Bootstrap

Web Design and Development, Collaborator

Visit UA Bootstrap

UA Bootstrap is the University of Arizona's flavor of Twitter Bootstrap, a popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

UA Profiles

Web Design, Collaborator

Visit UA Profiles

UA Profiles is a site where University faculty members can create a profile that displays information about teaching, research, service, and other activities. I designed the site using HTML, CSS, JQuery, and UABootstrap. I then passed it on to a developer who added PHP, JSON, and created the site using a Drupal environment.

Car Smart

Game Design, Interface, Script, Art Direction

View in iTunes store  |  View in Android store

Car Smart is an iPad based game which teaches young adults how to purchase their first vehicle. It is designed to be played in a classroom setting. My involvement includes the creation of the game in its intirety, in collaboration with an illustrator and a Java programmer.

Arizona Mobile

UI/UX, Icon Design

Arizona Mobile is the campus' mobile app created specifically to assist students. It shows class availabilty and location, keeps track of the schedule and GPS location of the CatTran, and many other features which help make student life easier.

REConnect

UI/UX Design, Icon Design

Currently not published

REConnect is a mobile app created for the University of Arizona's Recreational Center. It keeps track of and displays occupancy levels of the various areas in the Rec, as well as what equipment is currently available to rent, what rooms are available to use, and what events are going on at the Rec Center.

Ag Water

Icon Design

View in Android store

Ag Water was designed to aid in determining compliance with national Food Safety Modernization Act regulations and Leafy Green Marketing Agreement recommendations. My involvement includes the creation of the app icon and store banner.
